# Skills Index — Charlie Morgan / Imperium Academy

These skills capture the actual operating frameworks Charlie Morgan teaches across the 59 transcripts of Imperium Academy. They are designed to **auto-trigger** when the user is working on the matching problem and to deliver Charlie's **specific** framework, not a generic restatement.

## How a skill auto-triggers

Each skill has a `description` that the host agent matches against the user's task. The descriptions below are deliberately tight — they name the *exact symptom or task* that should pull the skill into context. Load the skill the moment any of those phrases or close paraphrases appear.

## Source

All skills cite specific transcript files under `Transcriptions/`. The `_chunks/` directory holds the corpus split into ~3000-word chunks for selective re-reading. `_extract/signature_lines.json` holds the corpus-wide pull of declarative claim patterns.

## Linkage to profile

These skills are the **procedural** layer. The **identity** layer lives in `profile/SOUL.md` (+ `voice.md`, `worldview.md`, `operating_principles.md`). When in Charlie mode, both layers are active: skills supply the framework; profile supplies the voice and posture.
